HTML

HTML (HyperText Markup Language) is used to give content to a web page and instructs web browsers on how to structure that content

Element and structure
Tag Name Description
<li> List Item Element to create list items inside ordered lists <ol> and Unordered lists <ul>
<video> Video Element to embeds a media player for video playback
<em> Emphasis Element to emphasis element emphasizes text and browsers will usually Italicize the emphasized text by default
<ol> Ordered List Element to creates a list of items in sequential order
<div> Div Element used as a container that divides an HTML document into sections and is short for "division"
Tables
<tr> Table Row Element used to add rows to a table before adding table data and table headings
<td> Table Data Element to add a cell of data to a table
<thead> Table Head Element defines the headings of table columns encapsulatedd in table rows
<tbody> Table Body Element semantic element that will contain all table data other than table heading and table footer content
<th> Table Heading Element used to add titles to rows and columns of a table and must be enclosed in a table row element
<tfoot> Table Footer Element uses table rows to five footer content or to summarize content at the end of a table

CSS

CCS or Cascading Style Sheets, is a language that is used in combination with HTML that customizes how HTML elements will appear. CSS can define styles and change the layout and design of a sheet
